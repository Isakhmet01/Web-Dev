# Python OOP Project

This project demonstrates basic Object-Oriented Programming in Python.

## Classes

### Animal (Base Class)
Attributes:
- name
- age
- color

Methods:
- eat()
- speak()
- __str__()

### Dog (Child Class)
Inherits from Animal.

Additional attribute:
- breed

Methods:
- speak() (overridden)
- guard()

### Cat (Child Class)
Inherits from Animal.

Additional attribute:
- indoor

Methods:
- speak() (overridden)
- climb()

## How to Run

Run the main file:

python main.py
